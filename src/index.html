Pertanyaan Umum

Berikut beberapa pertanyaan yang sering ditanyakan oleh pembaca.

Bagaimana cara menggunakan template ini?

Cukup download atau fork project ini, lalu modifikasi konten sesuai kebutuhan. Template ini sudah mendukung Tailwind dan dark mode bawaan.

Apakah ini bisa digunakan untuk proyek komersial?

Ya, template ini bersifat open source dan dapat digunakan untuk kebutuhan pribadi maupun komersial tanpa batasan.

Apakah bisa menambahkan fitur komentar?

Tentu. Anda bisa integrasikan komentar dummy atau pakai layanan seperti Supabase, Commento, atau sistem backend lainnya.
